Project Redesign

The website for the redesign is http://theatredowntown.net/index.html. It is a site for a local theater located in downtown, Orlano. This website needs a more organized look and feel to it. It will also need to have a color palette created for it to keep a consistent color theme throughout the site. Currently, the site looks different on each page. The navigation items are on multiple lines and look very outdated. Throughout this page, I will explain in detail what the redesign will be.

Main Goals

The main goals of this redesign are:

  1. Make the site more user friendly.
  2. Clean up the site.
  3. Organize the content.
  4. Set a consistent pattern between each page.

Target Audience

The main target audience for this site is females, between the ages of 25-34 years old. A large majority of these are graduate school graduates with no children. They also mostly view this site from either school or work. This is important to note, as at work or school they will be pressed for time and need to accomplish things fast.
